# game-qa-expert

> 游戏开发落地答疑 QA 专家项目
> 基于 H72 Skill V1.3 架构

---

## 项目定位

区别于冰冷的数据查询工具，融合问答反馈、设计评测、落地性评估等"人"的要素，成为真正懂开发、能判断、可学习的智能助手。

**核心定位：不是简单的"查数据"，而是"帮开发者解决问题"**

## 核心能力

| 能力类型 | 具体功能 |
|----------|----------|
| **查询** | 配置流程、表格、ST节点、编辑器文档、GM指令、负责人、Graph/MTH节点 |
| **生成** | 临时ST文件模板生成并发送到群内 |
| **评测** | 设计合理性评估、落地性判断 |
| **学习** | 基于用户反馈持续优化 |
