# 2026-04-09 关键洞察：Event 分类框架与会话状态管理

## Event 分类方法论
- **人文模型**：从市井生活习惯（占座、排队、围观）出发，逆向工程为游戏机制。
- **物理模型**：基于物理常识（光反射、重力、遮挡）设计的非数值解谜。
- **机制映射**：建立“现实动词”到“引擎组件”的 1:1 映射表，确保设计的技术可落地性。

## 会话状态管理 (Session State)
- **Token 经济学**：监控长对话中的 Token 消耗，在临界点（~20 轮）触发“状态压缩与持久化”。
- **RAG 标签自动生成**：在文档归档时，自动提取关键词（Keywords）并生成元数据，优化长线记忆的召回精度。
